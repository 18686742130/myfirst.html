CSS：
    样本重置：
         *{
            margin: 0;
            padding: 0;
        }
        ul{
            background: #ff0000;
            list-style: none;
        }
        li{
            background: #00ff00;
        }
    背景颜色：background-color:000000;
    塌陷问题：
        内容溢出时,溢出隐藏----overflow: hidden;
        设定高度--height:xxpx;
    定位：
        不可层叠（父元素）-- position: relative;      相对定位
        可层叠--position: absolute;      绝对定位
        定位后在子元素中：  top: 0;
                                   left: 0;
        只有定位的元素才能设置自己的位置（浮动的元素需要定位）
    透明度：
        opacity：（0-1）；  0-透明  1-100%
        为了解决兼容性问题

    上下层级关系：z-index:1;只有定位的元素才可以设置这个属性

    块元素居中--margin:xx px auto;  xx是数值 块上方外边距
    光标变成小手--cursor:pointer;       鼠标移入时变成小手
